# SansarPay Beta Launch

This file is for manually onboarding the first 10 SansarPay beta users.

Use it as a simple founder playbook:

- invite users on WhatsApp
- help them finish setup
- follow up until they complete the core actions
- track the first beta outcomes

Target users:

- Nepali workers in the Gulf
- users sending money home to Nepal
- users comfortable with WhatsApp

## Founder Workflow

For each beta user:

1. Send the first WhatsApp invite.
2. Wait for signup.
3. Check if the user logged the first transfer.
4. Check if the user set a savings goal.
5. Check if the user shared the family link.
6. Check if family opened the link.
7. Ask for feedback after first use.

Keep the process manual and simple for the first 10 users.

## Copy-Paste Founder Scripts

### 1. First WhatsApp Invite

```text
Namaste dai/bhai. I am testing a new simple app called SansarPay for Nepali workers who send money home.

It helps you:
- log remittance transfers
- track a savings goal
- share a family update link

I am onboarding only 10 beta users first.

If you are okay to test it, I will send you the link. It should take around 5 minutes to start.
```

### 2. Follow-Up After Signup

```text
Namaste. I saw you signed up in SansarPay.

Next step is simple:
1. log your latest transfer
2. set one savings goal

If anything is unclear, message me here and I will help.
```

### 3. Transfer Logging Reminder

```text
Namaste. Small reminder to log your first transfer in SansarPay.

Please add:
- amount
- transfer date
- transfer method

Once that is done, your dashboard will start showing progress.
```

### 4. Family Link Sharing Reminder

```text
Namaste. After you log your transfer, please open Settings in SansarPay and copy your family link.

Send that link to your wife, mother, father, or family member on WhatsApp so they can see your update.

If you want, I can help you do this step now.
```

### 5. Feedback Request

```text
Namaste. Thank you for trying SansarPay.

Can you send me quick feedback on these 3 things:
1. what was easy
2. what was confusing
3. what should be improved first

Short voice note or text is both okay.
```

## Troubleshooting Guide

### OTP Not Received

What to check:

- Confirm the phone number is entered in the correct Nepal mobile format.
- Ask the user to wait 1 to 2 minutes.
- Ask the user to try again once.
- Confirm the user has mobile signal and can receive SMS.

Founder reply:

```text
Please check the mobile number once again and try again after 1 minute.

If OTP still does not come, send me the number you are using and I will help check it with you.
```

### Phone Number Wrong

What to check:

- Confirm the number starts with Nepal mobile digits.
- Confirm the user did not add extra digits.
- Confirm the user is using their own active number.

Founder reply:

```text
Please enter your Nepal mobile number carefully once again.

Example format:
98XXXXXXXX

If you entered the wrong number before, go back and try again with the correct one.
```

### Family Link Not Opening

What to check:

- Confirm the full link was copied.
- Confirm the link starts with the real app domain, not localhost.
- Confirm the user sent the full link in WhatsApp.
- Ask the family member to open it in the browser again.

Founder reply:

```text
Please copy the family link again from Settings and send the full link again on WhatsApp.

If the page still does not open, forward the link to me and I will test it.
```

### WhatsApp Message Not Opening

What to check:

- Confirm WhatsApp is installed on the phone.
- Confirm the phone can open normal WhatsApp chats.
- Ask the user to copy the link manually if the share button fails.

Founder reply:

```text
If WhatsApp is not opening from the app, please copy the link manually and paste it into your family chat.

That is okay for beta. We only need the link to reach your family member.
```

## Success Metrics

Beta is on track when these numbers are reached:

- 10 users signed up
- 5 transfers logged
- 3 savings goals created
- 3 family links opened

## Daily Founder Review

At the end of each day, check:

1. How many users signed up today?
2. How many users logged a transfer today?
3. How many users created a savings goal?
4. How many users shared a family link?
5. How many family links were opened?
6. What problems were repeated by more than one user?

## Notes for the First 10 Users

- Keep support on WhatsApp.
- Use simple Nepali-English mixed language if needed.
- Do not send long explanations.
- Help the user complete one step at a time.
- If a user gets stuck, call or send a voice note instead of writing a long message.
